Projects

TechShop eCommerce Site using MERN Stack

This is an eCommerce web application designed to offer a seamless shopping experience. It features a full shopping cart, product reviews and ratings, a top products carousel, and advanced search functionality. Users can manage their profiles and view orders, while administrators have control over product, user, and order management, including marking orders as delivered. The checkout process is fully integrated with PayPal and credit card payment options. Additionally, the application includes a database seeder for easy setup of products and users. The frontend and backend are hosted on Render(free tier, so this may take a second to load).

Technologies used:

React, Redux, Node.js, Express, MongoDB, Render Hosting.

Face Recognition Web App

This is a web application built with React that allows users to register, sign in, and submit image URLs for analysis. The application utilizes a server-side backend application using Express.js and a PostgreSQL database. It handles user registration, authentication, profile management, and image recognition using the Clarifai face-detection AI model. The backend and database are hosted on Heroku.

Technologies used:

React, Node.js, Express, PostgreSQL, Heroku, Clarifai AI Model.

Fitness Mobile App

This is a fitness mobile application built with React Native. I worked collaboratively on a team with three other students to develop this app to inspire positive fitness habits, accountability, and social challenges that engage users in a fun and productive way with measurable fitness gains. It handles user registration, Two-Factor Authentication, customizable workouts and calendar tracking. All UI is built with React Native reusable components.

React Native, Node.js, CSS, Express, MySQL
