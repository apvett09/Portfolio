Projects

Face Recognition Web App

This is a web application built with React that allows users to register, sign in, and submit image URLs for analysis. The application utilizes a server-side backend application using Express.js and a PostgreSQL database. It handles user registration, authentication, profile management, and image recognition using the Clarifai face-detection AI model. The backend and database are hosted on Heroku.

Technologies used:

React, Node.js, Express, PostgreSQL, Heroku, Clarifai AI Model.
